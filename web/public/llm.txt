# mailrs - AI Email API

> AI agents can send, read, search, and reply to emails via MCP or REST API.

Base URL: https://mail.golia.ai
Auth: Bearer API key on all requests
Full API docs: https://mail.golia.ai/llm-full.txt

## MCP (recommended)

Streamable HTTP endpoint: https://mail.golia.ai/mcp
Tools: send_email, read_email, search_emails, reply_email, list_conversations

```json
{"mcpServers":{"mailrs":{"type":"streamable-http","url":"https://mail.golia.ai/mcp","headers":{"Authorization":"Bearer <API_KEY>"}}}}
```

## REST API

- POST /api/mail/send - send email (JSON)
- POST /api/mail/send-multipart - send with attachments (multipart/form-data)
- GET /api/conversations - list conversations
- GET /api/conversations?q=keyword - search
- GET /api/mail/messages/{uid} - read message
- POST /api/agent/webhooks - subscribe to new email events
