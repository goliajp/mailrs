## REST API

### Email
- POST /api/mail/send — send email (JSON)
- POST /api/mail/send-multipart — send with attachments
- GET /api/conversations — list conversations (?q=keyword, ?category=, ?limit=)
- GET /api/conversations/{thread_id} — thread messages
- GET /api/mail/messages/{uid} — read message
- POST /api/conversations/{thread_id}/read — mark read
- POST /api/conversations/{thread_id}/star — star
- POST /api/conversations/{thread_id}/archive — archive
- DELETE /api/conversations/{thread_id} — delete thread
- GET /api/mail/folders — list folders
- GET /api/contacts — search contacts

### Admin (requires permissions)
- GET/POST/DELETE /api/admin/domains — domain CRUD
- GET/POST/DELETE /api/admin/accounts — account CRUD
- GET/POST/DELETE /api/admin/aliases — alias CRUD
- GET/POST/DELETE /api/admin/email-groups — email group CRUD + /members
- GET/POST/DELETE /api/admin/groups — group CRUD + /permissions + /members
- GET/POST/DELETE /api/admin/apps — app CRUD + /scopes
- GET /api/admin/permissions — list all permissions
- GET/POST /api/admin/greylist/local-lists, DELETE /api/admin/greylist/local-lists/{id} — local greylist white/black CRUD (admin.greylist)
- POST /api/admin/reconcile-maildir — index maildir files missing from PG; {"dry_run": true} reports only (internal.rpc)

### Agent
- POST/GET/DELETE /api/agent/keys — API key management
- POST/GET/DELETE /api/agent/webhooks — webhook subscriptions

### Status
- GET /api/status, /api/health, /api/readiness
